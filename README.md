## Instruction:

### Requirements:
- Node version >= 14+

### How to run:
```sh
$ npm install
$ npm run build
Then
$ node ./dist/src/app.js [path to json file]
```
```
or for this project:
$ npx ts-node src/app.ts ./src/input.json

There is an equivalent command in package.json:
$ npm start
```
### Tests

To run the tests use the following command:

```sh
$ npm test
```
